Šta je Git i kako funkcionira?

Git je sistem za kontrolu verzija (VCS - Version Control System) koji omogućava praćenje promjena u fajlovima tokom vremena i kolaboraciju među programerima.
Princip rada Git-a je baziran na praćenju stanja fajlova, omogućavajući korisnicima da:

  • Snime trenutnu verziju projekta (commit).
  • Prate povijest svih izmjena.
  • Rade na različitim verzijama projekta putem grana (branch).
  • Spajaju promjene bez gubitka podataka.

Osnovno postavljanje Git repozitorijuma

Da biste počeli koristiti Git na svom računaru, potrebno je prvo da ga instalirate i podesite osnovne parametre kao što su ime i e-mail adresa.
Nakon toga možete kreirati novi repozitorijum i početi praćenje vaših projekata.
